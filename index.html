⚙️ Pengaturan

🌐 Kamus Bersama (Recommended)

Isi Worker URL & Token buat sync ke kamus bersama tim. Kalau punya Admin Token, kamu bisa edit/hapus juga. Nama akan muncul sebagai "ditambahkan oleh …" di kartu yang kamu tambah.

🤖 Gemini API (Solo Mode)

Ini hanya perlu diisi kalau Worker URL kosong — buat pakai app solo tanpa server bersama. Data disimpan lokal di HP. Kalau kamu sudah pakai Worker di atas, kosongkan bagian ini — key Gemini sudah di server.

✅ Pengaturan tersimpan permanen di HP kamu (LocalStorage). Setelah simpan sekali, kamu tidak perlu setup ulang — buka app kapan aja langsung jalan. Kalau ganti HP / clear browser data, baru perlu setup lagi.

💾 Backup & Restore

Kata-katamu otomatis tersimpan di browser HP ini. Backup ke file supaya kalau hapus browser tidak hilang, atau untuk share antar device.

📋 Clipboard (Backup Cepat)

Salin ke WhatsApp/Notes sebagai cadangan cepat waktu tidak sempat backup file.
